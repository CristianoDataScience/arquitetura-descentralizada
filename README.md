# Arquitetura Descentralizada — Embracon

Apresentação interativa sobre a arquitetura descentralizada da Embracon, incluindo slides sobre:

- **Visão geral** da arquitetura descentralizada
- **Pilares técnicos**: segurança, integração, dados e infraestrutura
- **Produtos Embracon** na nova arquitetura (RH, App do Cliente, Ta na Mão, Simpli+, Convert+)
- **FinOps & Governança de Dados**: controle de custos de nuvem, governança, liberdade para criar IA
- **Tecnologias** utilizadas na stack

## Como visualizar

Abra o arquivo `index.html` no navegador. Navegue entre os slides com os botões ou as teclas `←` `→`.

| Arquivo | Conteúdo |
|---------|----------|
| `index.html` | Apresentação principal da arquitetura |
| `finops.html` | Levantamento de FinOps, governança de dados e IA |
| `tecnologias.html` | Stack tecnológica detalhada |
